# Antigravity Starter Repo

Welcome to the **Antigravity Skills Starter Repo**! 

This repository is designed as a launchpad for developers working with Antigravity, providing example skills the agent can use to work on non-coding tasks.

## Overview

This project serves as a foundation for performing non-coding tasks on Antigravity.

## Features

- **Use Pre-built Skills**: Comes with a set of ready-to-use skills for common tasks.
- **Extensible Architecture**: Easily add your own custom skills to expand functionality.
- **Customizable**: Modify existing skills or the core structure to fit the specific needs of your project.

## Getting Started

1. **Explore Skills**: Navigate to `.agent/skills` to see how skills are structured.
2. **Create a Skill**: Add a new directory in `.agent/skills` with your skill definition and resources.
3. **Customize**: Edit any existing skill to better suit your workflow requirements.

Feel free to fork this repository and make it your own!
